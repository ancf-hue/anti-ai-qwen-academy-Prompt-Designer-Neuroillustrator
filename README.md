> 🎓 **10-дневный курс по этичному ИИ-арту** для библиотекарей, волонтёров и педагогов

## 🎯 Что это?

Образовательное веб-приложение **с офлайн-режимом**, которое обучает **Промпт-дизайнеров / Нейроиллюстраторов** для работы в библиотеках и культурных учреждениях.

**Возможности:**
- ✍️ **10 интерактивных уроков** с практическими заданиями
- 🧠 **Финальный экзамен** (10/10 для сертификации)
- 🛠️ **Инструменты**: Конструктор промптов, Чек-лист этики, Библиотека стилей
- 📚 **Библиотека знаний**: Документация роли, рабочие процессы, KPI
- 🔐 **Офлайн-сертификат** с SHA-256 верификацией
- 🌐 **Два языка**: Русский / English
- 🔊 **Голосовое сопровождение** для доступности
- 💾 **100% офлайн** — работает без интернета

## 🏆 Условия получения сертификата

- ✅ Пройти все 10 уроков
- ✅ Сдать экзамен на 10/10
- ✅ Задокументировать 5 кейсов в форматах Qwen

## 🎨 Для кого этот курс?

- 📚 **Библиотекари**, создающие визуальный контент для мероприятий
- 🧑‍🏫 **Педагоги**, обучающие ИИ-грамотности
- 🎨 **Волонтёры** культурных проектов
- 🌍 **НКО**, которым нужны этичные иллюстрации
- 🧑‍🎨 **Новички** в промпт-инженерии с фокусом на этику

## 🛠️ Технологии

- Чистый HTML5 / CSS3 / JavaScript (без фреймворков)
- LocalStorage для сохранения прогресса
- Web Speech API для голосового сопровождения
- CSP заголовки безопасности
- Адаптивный дизайн (мобильные устройства)
